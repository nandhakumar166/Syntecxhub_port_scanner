# Syntecxhub_port_scanner
TCP Port Scanner

A Python-based TCP Port Scanner built using socket programming to perform network reconnaissance and port enumeration. The tool scans a target host for open ports within a specified range, leveraging multithreading for faster and more efficient concurrent scanning. It provides detailed scan results, including open, closed, and timed-out ports, while implementing robust exception handling and logging for reliability.

Features
TCP port scanning using Python sockets
Scan a single host across a custom port range
Multithreaded scanning for improved performance
Detection of open, closed, and timed-out ports
Logging of scan results for later analysis
Exception handling for network and connection errors
Beginner-friendly project for learning networking and cybersecurity concepts
Technologies Used
Python
Socket Programming
Multithreading (threading)
Logging Module
Learning Outcomes
Understanding TCP/IP networking fundamentals
Socket programming and client-server communication
Concurrent execution using threads
Network reconnaissance techniques
Error handling and logging in Python
